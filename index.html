Portafolio

Los proyectos en los que he contribuido

Gestor de documentos

El gestor de documentos se desarrolla utilizando tecnologías como TypeScript (TS) y Angular para el frontend, Laravel para el backend, y MySQL como sistema de gestión de base de datos. Este sistema está diseñado para la Universidad Técnica de Ambato, enfocándose en la seguridad de los datos mediante el uso de encriptación AES, así como en la gestión de documentos de las secretarías.

Proyecto de tablas de amortización

El proyecto de tablas de amortización tiene como objetivo principal facilitar el cálculo y visualización de la amortización de préstamos o inversiones a través de una interfaz web. El sistema está desarrollado utilizando tecnologías web como PHP para la lógica del servidor, HTML para la estructura de las páginas, JavaScript para la interactividad del usuario y MySQL como sistema de gestión de base de datos

Facturación Electrónica

El proyecto de facturación electrónica se desarrolla utilizando tecnologías como Angular para el frontend, Laravel para el backend, TypeScript (TS) para mejorar la calidad del código en el lado del cliente, y MySQL como sistema de gestión de base de datos. La facturación electrónica es un sistema que permite la emisión, envío y gestión de facturas de manera electrónica, cumpliendo con los estándares y requisitos legales.
